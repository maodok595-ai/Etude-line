Installer Étude LINE

Ajoutez l'application à votre écran d'accueil pour un accès rapide !

📱

Installation sur iPhone

Choisissez votre navigateur :

1

Ouvrir le menu Partage

Appuyez sur le bouton Partage ⬆️ en bas de Safari

2

Défiler vers le bas

⚠️ Important : Faites défiler le menu vers le bas pour trouver l'option

3

Ajouter à l'écran d'accueil

Sélectionnez Sur l'écran d'accueil ➕ puis "Ajouter"

ℹ️ Note : Chrome sur iPhone ne permet pas l'installation directe. Vous devez ouvrir le site dans Safari.

1

Copier l'URL

Appuyez longuement sur la barre d'adresse et sélectionnez "Copier"

2

Ouvrir dans Safari

Ouvrez Safari et collez l'URL dans la barre d'adresse

3

Suivre les instructions Safari

Cliquez sur le bouton 🧭 Safari ci-dessus pour voir les étapes
